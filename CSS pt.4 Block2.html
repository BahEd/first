Свойство border.

Border - граница блочного тега и для некоторых строчных(img). толщина линии_стиль линии_цвет линии. solid-линия; dotted-точки; dashed-пунктир. Смещается внутрь на ширину значения, не изменяя общий размер блока. Можно задать границу только для одной стороны: border-bottom(left/right/top) или убрать одну из сторон границы в таком случае оставить общий border и добавить запись с нужной стороной: border-left: 0px (none) border-radius: 10px 10px 10px 10px; - радиус скругления углов (сверху справа снизу слева); 10px - для всех сторон. Работает и без border, с overflow: hidden;

Запись в VScode: bd

Свойство outline.

outline - запись как и border. Не изменяет внутреннее содержимое блока. Указывается только для всех сторон сразу. Со строчным тегом будет не прямоугольник, а огибать текст.

Свойство box-shadow.

box-shadow - сдвиг по гор._сдвиг по верт._радиус тени_расстояние тени_цвет тени; - тень блока. можно через запятую дописать внутреннюю тень "inset_гор_верт_радиус_расстояние_цвет". Можно использовать с border-radius.

Запись в VScode: box-shadow: 6px 4px 3px 3px rgba(5, 167, 48, 0.753), inset 0px 0px 1px 1px;

Свойство opacity.

opacity - прозрачность блока. 0 - полностью прозрачный; 1 - полностью видный; 0.5 - 50%. Не убирает блок из верстки при значении "0" как display: none. Всё так же можно взаимодействовать (допустим там была ссылка, место где она была будет кликабельно).

Свойство visibility.

visibility - предназначен для отображения или скрытия элемента. Значение по умолчанию "visible" - виден. Значение "hiddden" - блок исчезнет, но место, которое он занимал останется и при этом нельзя взаимодейстовать как с opacity: 0;
