Псевдоклассы
Link - псевдокласс :link используется для оформления ссылок, которые пользователь ещё никогда не открывал (в рамках текущего домена).

Синтаксис: селектор:псевдокласс{свойства}.

Hover - псевдокласс :hover срабатывает при наведение на элемент.
Visited - псевдокласс :visited добавляется ссылкам, по которым уже переходил пользователь. Если вы задаёте стили для разных состояний ссылок, то следует придерживаться определённого порядка в объявлении стилей: :link — :visited — :hover — :active. LVHA!
Active - псевдокласс :active срабатывает в момент нажатия.
Focus - псевдокласс :focus срабатывает когда наведен фокус на объект, чаще всего используется для тегов <form>.

Child - псевдокласс :child При помощи этих псевдоклассов можно удобно выбирать элементы по их порядковому номеру внутри родительского элемента.

:nth-child(){...} - в "()" указывается порядковый номер элемента, к которому можно обратиться.

:nth-child(odd){...} - обращение к нечетным элементам.

:nth-child(even){...} - обращение к четным элементам.
