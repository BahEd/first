Background
Управляет фоном элемента. Универсальное CSS - свойство, которое позволяет задать несколько параметров фона: цвет, установить фоновое изображение, задать начальную позицию фонового изображения настроить повторение фонового изображения, задать метод скролла, настройка масштаба изображения. Так же значением свойства background может быть указана градиентная заливка фона.
